# پنل ادمین تورنمنت تخته‌نرد

پنل سبک برای مدیریت تورنمنت تخته‌نرد با قالب **حذف دوگانه اصلاح‌شده**.

## تکنولوژی‌ها

- Next.js 15
- TypeScript
- TailwindCSS
- Supabase

## راه‌اندازی

1. **نصب وابستگی‌ها**
   ```bash
   npm install
   ```

2. **تنظیم Supabase**
   - پروژه Supabase ایجاد کنید
   - فایل `.env.local` بسازید و از `.env.local.example` کپی بگیرید
   - مقادیر `NEXT_PUBLIC_SUPABASE_URL` و `NEXT_PUBLIC_SUPABASE_ANON_KEY` را تنظیم کنید

3. **اجرای Migration**
   - در داشبورد Supabase، بخش SQL Editor را باز کنید
   - محتوای `supabase/migrations/001_initial_schema.sql` را اجرا کنید

4. **اجرای پروژه**
   ```bash
   npm run dev
   ```

## ساختار تورنمنت

- **راند اول**: مسابقات به‌صورت تصادفی تولید می‌شوند
- **براکت اصلی**: برنده‌ها به راند بعدی می‌روند
- **براکت بازنده‌ها**: بازنده‌ها یک راند اضافه بازی می‌کنند
- **بازگشت به اصلی**: برنده‌های براکت بازنده‌ها به براکت اصلی برمی‌گردند

## صفحات

| مسیر | توضیح |
|------|-------|
| `/dashboard` | نمای کلی، تعداد بازیکنان، مسابقات فعال، کنترل تورنمنت |
| `/players` | افزودن و حذف بازیکن |
| `/matches` | ثبت نتایج مسابقات |
| `/rounds` | نمایش راندها |
| `/bracket` | نمایش براکت اصلی و بازنده‌ها |

## جریان کار پیشنهادی

1. بازیکنان را در صفحه بازیکنان اضافه کنید
2. از داشبورد «ایجاد راند اول» را بزنید
3. «ایجاد راند بعدی (اصلی)» را بزنید (ساختار راند ۲ ایجاد می‌شود)
4. «ایجاد راند بازنده‌ها» را بزنید (بازنده‌های راند ۱ به این راند می‌روند)
5. نتایج مسابقات راند ۱ را ثبت کنید (برنده‌ها به راند ۲، بازنده‌ها به راند بازنده‌ها می‌روند)
6. برای هر راند جدید، مراحل ۳ تا ۵ را تکرار کنید
